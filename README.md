# Simple Authentication Website

A full-stack authentication system with login, signup, and dashboard.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite

## Features
- User Signup
- User Login
- Password Encryption
- Dashboard after Login
- Logout functionality

## How to Run
1. Clone the repository
2. Go to backend folder
3. Run `npm install`
4. Start server using `node server.js`
5. Open `frontend/auth.html` in browser
